# Global Travel Tours

A modern, responsive travel agency website for **Global Travel Tours** with a clean, professional design and high-quality travel imagery.

## Design

- **Primary:** Dark Blue `#0A3D62`
- **Accent:** Orange `#F39C12` (CTAs, highlights)
- **Background:** Off-white `#F5F6F7`
- **Typography:** Plus Jakarta Sans (body), Playfair Display (headings)

## Sections

1. **Hero** — Full-width travel background, headline, and CTA buttons
2. **Popular Destinations** — Paris, Dubai, Rome, Istanbul, Bali
3. **Tour Packages** — Package cards with duration, price, and Book Now
4. **Why Choose Us** — Best Destinations, Affordable Prices, Professional Guides, Easy Booking
5. **Travel Gallery** — Photo grid
6. **Testimonials** — Customer reviews with photos and star ratings
7. **Contact** — Form, phone (+252 905 565 070), email (engmiski@gmail.com), map placeholder
8. **Footer** — Social links, quick links, copyright

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
# Python 3
python -m http.server 8080

# Node (npx)
npx serve .
```

Then visit `http://localhost:8080` (or the port shown).

## Files

- `index.html` — Structure and content
- `styles.css` — Layout, colors, responsive styles
- `script.js` — Mobile menu and form handling

Images are loaded from Unsplash (CDN) for realistic travel photography.
